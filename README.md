# RAG_LLM_task_30_group_120
## Проект по разработке вопросно-ответной системы с использованием RAG
Учебный проект в рамках обучения в магистратуре "Искусственный интеллект" НИУ ВШЭ.
Группа ИСИН25. 1 Курс.

### Цель проекта: Создать рабочий пототип системы, которая способна находить релевантную информации в заданной коолекции документов и использовать ее для генерации точных и развернутых ответов на вопросы пользователя на основе RAG (Retrivel-Augmented Generation).

### План работ.
1. Формулировка задачи и выбор предметной области
2. Сбор даннных, формирование тестового набора данных
3. Постороение бейзлана системы
   3.1 Изучение архитектуры RAG, концепции векторных эмбеддингов, симантического поиска и работы с LLM
   3.2 Предобработка и векторизация данных
   3.3 Выбор векторной базы данных
   3.4 Реализация семантического поиска
   3.5 Интеграция с LLM
   3.6 Тестирование бейзлан модели
4. Проведение экпериментов по улучшению качества ответов системы
5. Продуктивизация модели (реализация чат-бота в Telegram).
